## Quick Start
### Prerequisites

- Python 3.12.x

### Guide
Run the following lines into your system's command prompt:
1. Clone the Repository:
```bash
git clone https://github.com/cayceee/Historical_Ship_Routes.git
```
2. Install uv if not already installed:
```bash
pip install uv
```
3. Sync uv:
```bash
uv sync
```
4. Run main.py:
```bash
uv run main.py
```
